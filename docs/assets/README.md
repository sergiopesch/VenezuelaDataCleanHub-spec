# Diagram Assets

These SVG diagrams are committed so the architecture docs render outside
Mermaid-aware Markdown viewers.

- `system-architecture.svg`: high-level service, policy, workflow, storage, and
  observability architecture.
- `hardened-ingestion-flow.svg`: source manifest, policy, ingestion, matching,
  quarantine, review, and promotion flow.
- `safety-boundaries.svg`: raw-record, derived-record, OpenClaw, review, and
  promotion safety boundaries.
- `production-gates.svg`: production release gates for API, identity, policy,
  storage, eventing, exports, biometrics, and observability.
